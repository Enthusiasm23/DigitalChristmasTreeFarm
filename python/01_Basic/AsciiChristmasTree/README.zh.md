## 概览
准备好让你的终端充满节日的欢乐吧！这个小小的Python脚本能够打印一个ASCII艺术风格的圣诞树，非常适合在命令行中增添一抹节日气氛。

## 如何享受
运行 `ascii_christmas_tree.py`，观看你的终端变身为文字版的冬日仙境，闪烁着基于文本的彩灯和欢乐！
