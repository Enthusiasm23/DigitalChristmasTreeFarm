## 项目概览
开始你的编程假日冒险吧，用这个Python脚本来搭建一棵节日气氛的圣诞树。非常适合编程初学者在他们的编程雪橇之旅上的一个温柔引导，这个脚本简单介绍了Python的turtle图形库的奇妙之处。

## 如何运行
想在你的屏幕上召唤这棵迷人的针叶树，只需在Python环境中执行`simple_christmas_tree.py`脚本，观看你的虚拟树如何生机勃勃地展现出来！
